# List of Algorithms implemented
## Sorting Algorithms
- Bubble Sort
- Merge Sort
- Quick Sort
- Insertion Sort
- Selection Sort
- Heap Sort
- Counting Sort
- Radix Sort
<br>
## Graph-related Algorithms
- Breadth First Search (BSF)
- Depth Fitst Search (DFS)


## Backend
To start the backend service(`Port = 5000`), run:
`python3 app.py`

## Frontend
To start the frontend service(`Port = 3000`), run:
`npm start`
